# 指标
##  气候变化指数 （CCI）
- 一个国家的气候特征，如温度、降水或海平面因经度、纬度和海拔高度而有很大差异。
- 因此，我们定义年温度异常、年降水量异常、平均海拔高度
- 用气温、降水的年标准偏差（门限指数）和平均海拔高度来描述气候变化
